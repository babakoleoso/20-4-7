Cast 20-4 Process

This teaching page displays the 20-4 cowrie cast structure using the hierarchy of Eldest, Elder, Senior, and Junior. The cast is read from the closest four elements to the furthest four elements.

20-4 Cowrie Grouping Diagram

Image file used below: EldestCowrie2.png

Single Cast Method

Eldest

The closest four elements establish the primary Odu of the cast.

Elder / Senior / Junior

The next levels determine whether the Eldest is respected or challenged.

Ibi to Ire

When Eldership is respected, Ibi may be transformed into Ire.

20-4 Process Summary

  1. Cast 20-4 only once.
  2. Identify the closest four elements as the Eldest.
  3. Count the next four as the Elder group.
  4. Count the next four as the Senior group.
  5. Count the furthest four as the Junior group.
  6. Determine the Eldest Odu by orientation and direction.
  7. Ask whether the Eldership of the Eldest is respected by the next levels.
  8. If respected, the matter moves toward Ire.
  9. If not respected, the matter remains in Ibi until transformed.
